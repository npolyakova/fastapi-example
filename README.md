**Практическая работа №3**

1. Склонируйте этот проект и запустите приложение:
    1. Установите зависимости 
      ```pip install uvicorn fastapi```
    2. Запустите код ```python main.py```
    3. Перейдите на http://localhost:8002/ **(не 0.0.0.0:8002!)**, если видите message со строки 21 файла main.py, всё получилось
    4. Откройте Swagger - http://localhost:8002/docs

2. Измените API, согласно вашему порядковому номеру в списке группы. Задания находятся в task.txt
3. Для удобства выполнения CRUD операций, вынесите работу с данными в отдельный JSON файл.
4. Для GET запроса добавьте обработку параметра (parameter) sorting. Если его значение asc, сортируйте список от А до Я, если desc - наоборот.
5. Проведите тестирование своего функционала в Swagger. Отправьте запрос на POST/PUT/DELETE, а затем запрос на GET,
сделайте скриншоты обоих запросов, на них должно быть видно, что объект создан и отображается в списке объектов в результате гет-запроса.
6. Проведите тестирование с помощью инструмента POSTMAN. Отправьте GET-запрос с разными способами сортировки. Сделайте скриншоты обоих запросов.
7. Запушьте код в отдельную ветку гитхаб репозитория, который был создан для прошлого домашнего задания. Откройте пулл-реквест, вложите в него скриншоты из пункта 4 и 5.
8. В ответ на задание приложите ссылку на созданный пулл-реквест.
